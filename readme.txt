Display a "NEW!" badge for recently added WooCommerce products, making them stand out in your store.

== Description ==

**Product Spotlight Badge** is designed to boost the visibility of newly added products in your WooCommerce store. This plugin allows you to add a customizable "NEW!" badge on products, helping them stand out and attract customers' attention. Customize the badge with different text, text color, and background color, and specify the period for which the badge should appear.

- Enhance the visibility of new products with a customizable "NEW!" badge.
- Easily configure badge settings from the WordPress dashboard.
- Seamlessly integrates with WooCommerce-compatible themes.

== Installation ==

= Plugin Installation Method: =
1. Go to the WordPress dashboard.
2. Navigate to **Plugins > Add New**.
3. Search for "Product Spotlight Badge".
4. Click on the **Install** button.
5. Activate the plugin after installation.

= Installation via Zip File: =
1. Download the **Product Spotlight Badge** plugin zip file.
2. Go to **Plugins > Add New > Upload Plugin**.
3. Upload `product-spotlight-badge.zip` and install it.
4. Activate the plugin.

= Plugin Configuration: =
1. Navigate to **Settings > Product Spotlight Badge** in your WordPress dashboard.
2. Customize the badge text, colors, and display duration according to your preference.

== Frequently Asked Questions ==

= How do I customize the badge text and colors? =
You can customize the badge text, text color, background color, and duration from the Product Spotlight Badge settings page in the WordPress admin area.

= Does this plugin work with all themes? =
This plugin is designed to work with WooCommerce-compatible themes. If the badge does not display correctly, it might be due to theme-specific CSS conflicts.

= Can I display the badge on products that are not new? =
Currently, the badge is meant for newly added products. Future updates may provide more options for customization.

= How long will the badge be displayed? =
You can set the number of days the badge should be displayed for new products via the plugin settings.

= Is there any impact on site performance? =
The Product Spotlight Badge plugin is lightweight and optimized to ensure it does not affect your site’s performance.

= Can I display the badge on product pages as well? =
Yes, you can configure the plugin to show the badge both in the shop loop and on individual product pages.
